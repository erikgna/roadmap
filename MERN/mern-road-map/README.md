# Este é um APP feito usando REACT e NODE

## Para rodar o BACKEND

Você precisa abrir a pasta "mern-road-map", de preferência usando VSCode, em seguida, abra um terminal e digite o seguinte comando "cd backend" e então digite "nodemon server". É necessário ter o NodeJs instalado no PC. Talvez seja necessário instalar o módulo "nodemon" (npm install nodemon) no terminal. 

## Para rodar o FRONT-END

É necessário abrir um novo terminal e digitar "npm start".

## Talvez seja necessário instalar os packages.

Para instalar digite o seguinte código no terminal da pasta "mern-road-map": "npm i axios bootstrap react reac-datepicker react-dom react-router-dom react-scripts web-vitals"
